### INSTALLATION

```bash
# 1. Remove old curl
brew uninstall curl

# 2. Install modern curl
brew install curl

# 3. Add Homebrew curl to Fish PATH
fish_add_path /opt/homebrew/opt/curl/bin

# 4. Reload Fish shell
exec fish

# 5. Verify active curl binary
which curl
/opt/homebrew/opt/curl/bin/curl

# 6. Verify build features
```

### VERIFICATION

```bash
# VERIFY HTTP/2
test (curl -o /dev/null -s -w "%{http_version}" https://google.com) = "2"; and echo "✅ HTTP/2 ACTIVE"; or echo "❌ HTTP/2 FAILED"

# VERIFY HTTP/3
test (curl --http3 -o /dev/null -s -w "%{http_version}" https://cloudflare-quic.com) = "3"; and echo "✅ HTTP/3 ACTIVE"; or echo "❌ HTTP/3 FAILED"

# VERIFY FORCED HTTP/3
test (curl --http3-only -o /dev/null -s -w "%{http_version} %{response_code}" https://cloudflare-quic.com) = "3 200"; and echo "✅ HTTP/3 200 OK"; or echo "❌ HTTP/3 FAILED"

# VERIFY ALT-SVC CACHE GENERATION
curl --alt-svc ~/.curl_alt_svc -o /dev/null -s -I https://cloudflare.com; and test -s ~/.curl_alt_svc; and echo "✅ ALT-SVC CACHE OK"; or echo "❌ ALT-SVC FAILED"

# VERIFY ALT-SVC CACHE CONTENT
grep -q "h3 cloudflare.com 443" ~/.curl_alt_svc; and echo "✅ ALT-SVC H3 OK"; or echo "❌ ALT-SVC FAIL"

# Inspect cache:
cat ~/.curl_alt_svc
# Your alt-svc cache. https://curl.se/docs/alt-svc.html
# This file was generated by libcurl! Edit at your own risk.
# h2 cloudflare.com 443 h3 cloudflare.com 443 "20260512 09:37:37" 0 0
# h3 cloudflare.com 443 h3 cloudflare.com 443 "20260512 10:04:57" 0 0

# VERIFY AUTOMATIC HTTP/3 UPGRADE
test (curl --alt-svc ~/.curl_alt_svc --http3-only -o /dev/null -s -w "%{http_version} %{response_code}" https://cloudflare.com) = "3 301"; and echo "✅ HTTP/3 AUTO-UPGRADE OK"; or echo "❌ HTTP/3 AUTO-UPGRADE FAIL"

# VERIFY BROTLI SUPPORT
curl -H "Accept-Encoding: br" -sD - https://www.cloudflare.com -o /dev/null | grep -qi "content-encoding: br"; and echo "✅ BROTLI OK"; or echo "❌ BROTLI FAIL"

# VERIFY ZSTD SUPPORT
curl -H "Accept-Encoding: zstd" -sD - https://www.facebook.com -o /dev/null | grep -qi "content-encoding: zstd"; and echo "✅ ZSTD OK"; or echo "❌ ZSTD FAIL"

# VERIFY TLS 1.3
curl -v https://google.com 2>&1 | grep -q "TLSv1.3"; and echo "✅ TLS 1.3 OK"; or echo "❌ TLS FAIL"

# VERIFY POST-QUANTUM TLS
curl --http3 -I -v https://cloudflare-quic.com 2>&1 | grep -q "X25519MLKEM768"; and echo "✅ PQ TLS OK"; or echo "❌ PQ TLS FAIL"
```

### HTTP/3 NOTES

**HTTP/3 uses:**

* QUIC transport
* UDP instead of TCP
* multiplexed streams
* lower handshake latency
* reduced head-of-line blocking

> curl automatically upgrades connections using: alt-svc and cache when available.
